Создать каталог товаров

Сущности:
1. Категории
    - У категории есть название
    - У каждой категории есть массив атрибутов
    - У атрибута есть название и массив значений.

2. Товар:
    - У товара есть название
    - У товара есть дата создания
    - У товара есть категория
    - При выборе категории мы так же назначить для товара атрибуты из этой категории
    - Значения для каждого атрибута должны быть только те, которые были назначены для атрибуту при создании категории.
